Météo App

Cette application est un outil qui permet d’obtenir les prévisions climatiques de la majorité des villes du monde en temps réel. Il vous suffit de lanceer la recherche. Pour le développement de cette application j’ai utiliser le langage javascript et la méthode «FETCH» pour extraire les données des API. J’ai en effet combiné deux différents API, un permettant d’obtenir les données climatiques à partir de coordonnées géographiques et un autre pour obetenir les coordonnés géographique des villes à partir de la valeur saisie dans la barre de recherche.
